# Pokédex
Description: 
This allows the user to query the pokémon API at https://pokeapi.co and prints out the information in a clean way. The users can search for a pokémon, item, or move and get a easy-to-read information about what they searched for.
# To Use:
Type *node ./pokedex.mjs* to start the program, then type either 1, 2, or 3 to search for a pokémon, item, or move.
